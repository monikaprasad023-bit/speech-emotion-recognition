# Speech Emotion Recognition using Deep Learning

## 📌 Description
This project identifies human emotions from voice input using deep learning techniques.

## 🧠 Technologies Used
- Python
- TensorFlow
- Keras
- LibROSA
- NumPy

## 📊 Dataset
- RAVDESS (Ryerson Audio-Visual Database of Emotional Speech and Song)

## ⚙️ Features
- Detects emotions from speech signals
- Uses audio feature extraction
- Deep learning-based classification
